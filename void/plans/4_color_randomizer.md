# Генерация красивых цветовых сочетаний

## Текущая реализация
- Простой рандомайзер: полностью случайные цвета для Type, Back, Grid
- Кнопка Random на панели View

## Варианты улучшения

### 1. Теория цвета (HSB-based)
Генерация на основе цветового круга:
- **Комплементарные** — два противоположных цвета (hue + 180°)
- **Триадные** — три цвета с равным интервалом (hue + 120°)
- **Аналоговые** — соседние оттенки (hue ± 30°)
- **Split-complementary** — основной + два цвета рядом с противоположным

### 2. Open-source библиотеки

- **[Chroma.js](https://gka.github.io/chroma.js/)** — мощная библиотека для работы с цветами, генерация палитр, интерполяция
- **[TinyColor](https://github.com/bgrins/TinyColor)** — легковесная, умеет генерировать комплементарные, аналоговые, триадные цвета
- **[Color Thief](https://lokeshdhakar.com/projects/color-thief/)** — извлечение палитры из изображений
- **[Culori](https://culorijs.org/)** — современная библиотека с поддержкой множества цветовых пространств

### 3. API сервисы

- **[Colormind](http://colormind.io/api-access/)** — AI-генератор палитр, обучен на реальных дизайнах
- **[Huemint](https://huemint.com/)** — ML-based генератор для UI
- **[Coolors API](https://coolors.co/api/)** — генерация трендовых палитр

### 4. Простой алгоритм для контраста

Можно сделать так: фон — случайный цвет, текст — контрастный (светлый фон → тёмный текст), сетка — полупрозрачный вариант текста.

## Реализация Color Chaos

Color Chaos — режим, где каждый модуль каждой буквы получает свой уникальный цвет.

### Подходы:

1. **Полная рандомизация** — каждый модуль = случайный цвет
2. **Градиентная рандомизация** — цвета генерируются с учётом позиции модуля (создаёт плавные переходы)
3. **Палитра-основа** — генерируется базовая палитра (3-5 цветов), затем каждый модуль случайно выбирает из неё
4. **HSB-вариации** — выбирается базовый цвет, затем для каждого модуля варьируются насыщенность и яркость

### Техническая реализация:

- Добавить параметр `colorChaos` в настройки
- В `VoidRenderer` при рендеринге модулей проверять `colorChaos`
- Если включен — генерировать цвет для каждого модуля индивидуально
- Кэшировать цвета для стабильности (чтобы при перерисовке цвета не менялись)
- При нажатии Update — перегенерировать все цвета
