Void is a modular generative type system and a web-based tool for designing expressive typography. Unlike a conventional font, Void exposes the construction logic behind the typeface: weight, line count, rhythm, color, effects, and randomness can all be adjusted directly in the browser.
The project began as a Processing sketch during the Creative Coding course at Werkstatt School, and was later rewritten and expanded into an open interactive tool. It supports basic Latin and Cyrillic character sets, as well as graphics export to SVG.
